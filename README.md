# galenos.pro — Frontend

Vite + React starter with ES/EN, profile (alias/real + flag) and patients (PDF upload).

## Env
Create `.env` in project root:
```bash
VITE_API_URL=https://YOUR_BACKEND_URL
```

## Run
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
```

Deploy on Vercel → Framework: Vite, Output: `dist/`.
